# number-game-plus

little Number Guess Game with some upgrades.

## Goal

A playground to experement and improve my Rust knowledge.

## Features

- Number guess game
- Different difficulties
- Saving Statistic

## Instalation

In case you want to try this:

```bash
git clone https://github.com/matgo1/number-game-plus.git
cd number-game-plus
cargo build --release
./target/release/number-game-plus
```
